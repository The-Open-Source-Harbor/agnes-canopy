# Baltimore Dashboard GitHub Package

## Run locally

```bash
pip install -r requirements.txt
streamlit run Baltimore_Dashboard_Updated.py
```

Or on Windows, double-click `RUN_LOCAL.bat`.

## Streamlit Cloud settings

- Repository: your GitHub org repo
- Branch: `main`
- Main file path: `Baltimore_Dashboard_Updated.py`

## Included files

- `Baltimore_Dashboard_Updated.py` (main app)
- `data/integrated/baltimore_integrated_with_places_2022.csv` (primary data)
- `data/integrated/baltimore_integrated_expanded_2022.csv` (fallback data)
- `.streamlit/config.toml`
- `requirements.txt`
- Documentation files in this folder
